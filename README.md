# My Portfolio CRUD

Dynamic data:

- Skills
- Experience
- Contact

Stack:

- Express
- SQLite
- Express session
- bcryptjs
- HTML + Tailwind CDN

Run:

```bash
cp .env.example .env
npm install
npm start
```

Admin:

```text
/admin.html
```

Default login follows `.env`:

```text
ADMIN_USERNAME=admin
ADMIN_PASSWORD=ChangeMe123!
```
